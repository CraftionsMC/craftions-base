
# craftions-base

This is the base of all craftions web apps.


## Run Locally

Clone the project

```bash
  git clone https://github.com/CraftionsMC/craftions-base
```

Go to the project directory

```bash
  cd craftions-base
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn express:live-start
```
